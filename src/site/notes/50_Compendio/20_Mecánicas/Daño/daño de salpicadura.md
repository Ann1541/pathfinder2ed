>[! tip] Resumen
>- El daño de salpicadura se aplica siempre que no sea fallo critico
>- No se suma el bonificador de fuerza ni destreza **al daño** (tirada puede aplicarlo)

Cuando usas un arma arrojadiza con este rasgo, **no sumas tu modificador por Fuerza** al daño. 
El daño de salpicadura se suma al daño inicial contra el objetivo principal *antes de aplicar debilidades o resistencias.*

### Reglas de Aplicación
- **Área:** Todas las criaturas a **5 pies (1,5 m)** o menos del objetivo (incluido el objetivo y el atacante, si está en rango) sufren el daño indicado.
- **En Impacto Crítico:** El daño de salpicadura **no se multiplica**.
- **Según el resultado del ataque:**
	- **Éxito o Éxito Crítico:** El objetivo y las criaturas en el área reciben el daño.
	- **Fallo:** El objetivo **sigue sufriendo** el daño de salpicadura (las criaturas adyacentes no).
	- **Fallo Crítico:** Nadie sufre daño.

> [!REGLA]
> El daño de salpicadura es ideal para activar **Debilidades**, ya que se aplica incluso si fallas el ataque (pero no si fallas críticamente).
