# [[50_Compendio/80_Acciones/Acciones de Habilidad generales/Descifrar escritura\|Descifrar escritura]]

**[[50_Compendio/80_Acciones/Acciones de Habilidad generales/Descifrar escritura\|DESCIFRAR ESCRITURA]] ([[Entrenado\|ENTRENADO]]) [[CONCENTRAR\|CONCENTRAR]] [[EXPLORACIÓN\|EXPLORACIÓN]] [[SECRETO\|SECRETO]]**

> [!recite]+ Informacion.
> - Cuando te encuentres con textos particularmente arcaicos o esotéricos, el DJ podría requerir que Descifraras la escritura antes de poder entenderla. Debes estar [[Entrenado\|entrenado]] en la [[Habilidad\|habilidad]] relevante para [[50_Compendio/80_Acciones/Acciones de Habilidad generales/Descifrar escritura\|Descifrar escritura]]. [[Arcanos\|Arcanos]] se utiliza típicamente para escribir acerca de la magia o la ciencia, [[Ocultismo\|Ocultismo]] para textos esotéricos acerca de misterios y filosofía, [[Religión\|Religión]] para escrituras sagradas y [[Sociedad\|Sociedad]] para mensajes en clave o documentos arcaicos.
> ---
> - Intentas descifrar una escritura o literatura complicada sobre un tema oscuro. Esto suele requerir de 1 minuto por página de texto, pero podría durar más (típicamente 1 hora por página para desencriptar mensajes cifrados o cosas parecidas). 
> - El texto debe estar en un [[Idioma\|idioma]] que puedes leer, aunque el DJ podría permitir que intentaras descifrar un texto escrito en un idioma poco familiar utilizando en su lugar [[Sociedad\|Sociedad]].

La [[CD\|CD]] la determina el DJ basándose en el estado o en la complejidad del documento, y podría requerir que hicieras una prueba para un texto corto o una prueba para cada sección de un texto más largo.

- **[[Éxito Crítico\|Éxito crítico]]** Entiendes el verdadero significado del texto. 
- **[[Éxito\|Éxito]]** Entiendes el verdadero significado del texto. Ahora bien, si se trataba de un documento en clave, captas el sentido en general pero podrías no conseguir una traducción palabra a palabra. 
- **[[Fallo\|Fallo]]** No puedes entender el texto y sufres un [[Penalizador por circunstancia\|penalizador -2 por circunstancia]] a las pruebas posteriores para descifrarlo. 
- **[[Fallo Crítico\|Fallo crítico]]** Crees que entiendes el texto de dicha página, pero en realidad has entendido mal lo que dice.

--------------------------------------------------------------------------------

Ejemplos de tareas de Descifrar

• **[[Entrenado\|Entrenado]]** tratado de filosofía básica

• **[[Experto\|Experto]]** código complejo, como por ejemplo una clave

• **[[Maestro\|Maestro]]** el código de un maestro de espías o notas de investigación avanzadas

• **[[Legendario\|Legendario]]** texto planario esotérico escrito en forma de metáfora por un antiguo celestial
