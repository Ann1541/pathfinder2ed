# [[50_Compendio/30_Recursos/Objetos/Sustancias alquímicas/Elixires Alquímicos/Elixires estándar/Elixir de ojo de águila\|Elixir de ojo de águila]]

**ELIXIR DE OJO DE ÁGUILA SUSTANCIA 1 O MÁS** 
[[Alquímico\|Alquímico]] [[Consumible\|Consumible]] [[Elixir\|Elixir]] 
**Uso** sostenido en una mano 
**Impedimenta** L 
**Activar** `pf2:1` [[Interactuar\|Interactuar]]

>[!recite]+ Informacion.
> Tras beber este [[Elixir\|elixir]], te das cuenta de detalles visuales sutiles. Durante la siguiente hora, obtienes un [[Bonificador por objeto\|bonificador por objeto]] a las pruebas de [[Percepción\|Percepción]], que es superior cuando intentas encontrar puertas secretas y [[Trampas\|trampas]]. Siempre que estés cerca de una puerta secreta o una trampa que podrías notar, el DJ tira secretamente una prueba de [[Percepción\|Percepción]] para ver si la encuentras.

|Menor|1|4 po|+1|+2|
|Moderado|5|27 po|+2|+3|

---

• **Tipo menor; Nivel 1; Precio 4 po** El [[Bonificador por objeto\|bonificador]] es +1 (o +2 para puertas secretas y [[Trampas\|trampas]]).

• **Tipo moderado; Nivel 5; Precio 27 po** El [[Bonificador por objeto\|bonificador]] es +2 (o +3 para puertas secretas y [[Trampas\|trampas]]).

• **Tipo mayor; Nivel 10; Precio 200 po** El [[Bonificador por objeto\|bonificador]] es +3 (o +4 para puertas secretas y [[Trampas\|trampas]]).

• **Tipo superior; Nivel 16; Precio 2.000 po** El [[Bonificador por objeto\|bonificador]] es +4 (o +6 para puertas secretas y [[Trampas\|trampas]]).
